# Digital-Diary

The diary will provide the following services.

1) Contacts
2) Calender
3) World Clock
4) Games
i) Snake and Ladder
ii) Maze Runner
iii) Tic Tac Toe
iv) Snake Mania
5) Horoscope
6) Converter
7) Calculator

Each of these features will have a seperate classe and will be finally embedded in to the main class.
